DINGO Guide and Changelog

This page is serves as a guide and changelog to the DINGO game odds model, whose page can be found here.

DINGO (Daily Individual Game Odds) is Baseball Analytica's model to predict MLB game odds for any given day in the baseball season. The model itself, which is proprietary, was created by Addison Kline in 2024 and has been in use since May 1 of that season.

How are game odds calculated?

For each individual game in its forecast, the DINGO model analyzes several metrics for both the home and away team to evaluate their ability to score and prevent runs. Then, it analyzes tens of thousands of previous MLB matchups, finding the matchups most similar to the current one and calculating probabilities based on a sample taken.

Can I use these projections?

If you reference or make use of these projections in any public way, we ask that you credit this site, providing a link if possible.
